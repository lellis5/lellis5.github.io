HTML Versions

HTML 1.0

This was the first HTML and was made by tim berners in 1991 but it wasnt released until 1993.

This version layed the groundwork for web pages with basic features.

HTML 2.0

HTML 2.0 was published as RFC 1866 and was made by the HTML working group of the internet engineering task force with dan connolly as the lead editor.

It was made november 24 1995 and it slowly added new features over the years it was used.

HTML 3.0/3.2

People tried to make 3.0 but it failed because it wasnt as practical as 3.2.

XHTML

W3C decided to shift to an XML based HTML equivalent which became XHTML.

HTML 4.0

It was made by W3C in 1997 and it was a big improvement from the last ones but it was shortly replaced by 4.01

HTML 4.01

HTML 4.01 was an improvement of 4.0 and became an official standard in december 1999.

It was the mainly used version in the early 2000's but it has since been replaced.

XHTML 1.0

It was completed in 2000 and it was a combination of html 4.01 in XML.

HTML5

Published in 2012 and was seen as an extended version of html 4.01.

This is the newest version of html and the most commonly used one
